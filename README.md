# Shop Automation Addon

Tự động hóa quy trình mua hàng cho Meteor Client 1.21.11

## ✨ Tính Năng

1. ✅ Gửi `/shop`
2. ✅ Chọn mục địa ngục
3. ✅ Chọn Que quỷ lửa
4. ✅ Mua 64 vật phẩm
5. ✅ Ấn nút mua
6. ✅ Thoát khỏi menu
7. ✅ Gửi `/order`
8. ✅ Chọn vật phẩm đúng
9. ✅ Đưa hàng vào inventory
10. ✅ Lặp lại tự động

## ⚙️ Cấu Hình

| Tùy Chọn | Mô Tả | Mặc Định |
|---------|-------|---------|
| `item-name` | Tên vật phẩm | Que quỷ lửa |
| `buy-times` | Số lần mua | 1 |
| `delay` | Độ trễ (ms) | 500 |

## 📥 Cài Đặt

1. Download `.jar` từ Releases
2. Đặt vào `.minecraft/addons/`
3. Restart game
4. Mở Meteor → Modules → Shop Automation → Enable

## 🎮 Cách Sử Dụng

1. Enable module `shop-auto`
2. Đứng gần NPC shop
3. Addon sẽ chạy tự động
4. Xem chat log để kiểm tra

## ⚠️ Lưu Ý

- Delay nên >= 500ms
- Test với `buy-times: 1` trước
- Tên vật phẩm phải chính xác với trong game
- Không chạy 24/7 để tránh bị ban
